## Technology

Here are the technologies used in this project.

- React JS
- React Icons
- React Router Dom
- Tailwind CSS
- Vite
- Three
- EmailJS

## Services Used

- Github
- Vercel

## Getting started

- Dependency

  - Git
  - NPM

- To clone and run the project:

  > $ git clone https://github.com/jonasmkunzler/jonask-portfolio

  > $ cd jonask-portfolio

  > $ npm i

  > $ npm run dev
